# ZOOP-cvičenia

Zdrojové súbory ku školským mini-zadaniam z predmetu _Základy objektovo-orientovaného programovania_ v programovacom jazyku __Java__. Tento školský predmet sa zameriava na základy objektovo-orientovaného programovania: pojem objektu, triedy, agregácie, zapuzdrenia, rozhraní a dedenia.
